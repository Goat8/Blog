# Blog

1. Microservices created in expressJs 
2. Async communicatin using event bus between microservices
3. Small react App for CRUD operation

   
### Features
1. Create Posts
2. List Posts
3. Create Comments
4. List Comments
5. Tag Comments

##### Services
1. Posts Service
2. Comments Service
3. Query Service
4. Moderation Service

##Tools
NodeJs ExpressJs, ReactJs
